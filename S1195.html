Java Language Specificationによる：

旧バージョンのJava SEプラットフォームとの互換性のため、

配列を返すメソッドの宣言では、配列型の宣言を構成する空括弧の組を正式なパラメータリストの後に（一部または全部）置くことが許可されています。

この陳腐化した構文は、新しいコードに使うべきではない。

コンプライアンス違反のコード例

public int getVector()[] { /* ... */ }    // Noncompliant

public int[] getMatrix()[] { /* ... */ }  // Noncompliant

コンプライアンス・ソリューション

public int[] getVector() { /* ... */ }

public int[][] getMatrix() { /* ... */ }
