親クラスのメソッドをオーバーライドすると、オーバーライドするメソッド内で明示的にsuper を呼び出さない限り、そのメソッドが呼び出されることはありません。場合によっては、super メソッドを呼び出さないことが許容されますが、JUnit 3TestCase のsetUp とtearDown の場合はそうではありません。

コンプライアンス違反のコード例

public class MyClassTest extends MyAbstractTestCase {

  private MyClass myClass;
    @Override
    protected void setUp() throws Exception {  // Noncompliant
      myClass = new MyClass();
    }

コンプライアンス・ソリューション

public class MyClassTest extends MyAbstractTestCase {

  private MyClass myClass;
    @Override
    protected void setUp() throws Exception {
      super.setUp();
      myClass = new MyClass();
    }
