メソッド内にreturn文が多すぎると、return文に遭遇するたびに実行の流れが途切れてしまうため、メソッドの本質的な複雑さが増してしまいます。このため、メソッドのロジックを読んで理解することが難しくなります。

コンプライアンス違反のコード例

デフォルトの閾値は3です：

public boolean myMethod() { // Noncompliant; there are 4 return statements
  if (condition1) {
    return true;
  } else {
    if (condition2) {
      return false;
    } else {
      return true;
    }
  }
  return false;
}
