コーディング規約を共有することで、チームでの共同作業を効率的に行うことができます。このルールは、すべての定数名が指定された正規表現に一致するかどうかをチェックします。

コンプライアンス違反のコード例

デフォルトの正規表現で^[A-Z][A-Z0-9]*(_[A-Z0-9]+)*$ ：

public class MyClass {
  public static final int first = 1;
}

public enum MyEnum {
  first;
}

コンプライアンス・ソリューション

public class MyClass {
  public static final int FIRST = 1;
}

public enum MyEnum {
  FIRST;
}
