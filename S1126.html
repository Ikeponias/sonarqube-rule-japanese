if-then-else に包まれた boolean リテラル文の返却を簡略化する必要があります。

同様に、ブーリアンリテラルのみが異なるif-then-else に包まれたメソッド起動は、単一の起動に簡略化されるべきである。

コンプライアンス違反のコード例

boolean foo(Object param) {
  if (expression) { // Noncompliant
    bar(param, true, "qix");
  } else {
    bar(param, false, "qix");
  }

  if (expression) {  // Noncompliant
    return true;
  } else {
    return false;
  }
}

コンプライアンス・ソリューション

boolean foo(Object param) {
  bar(param, expression, "qix");

  return expression;
}
