Java Language Specificationによる：

無名パッケージは、主に小規模または一時的なアプリケーションを開発する際や、開発を始めたばかりの時に便利なように、Javaプラットフォームによって提供されます。

このベストプラクティスを実施するために、Java 1.4以降、デフォルトパッケージにあるクラスは、名前付きパッケージからアクセスすることができなくなりました。

コンプライアンス違反のコード例

public class MyClass { /* ... */ }

コンプライアンス・ソリューション

package org.example;

public class MyClass{ /* ... */ }
