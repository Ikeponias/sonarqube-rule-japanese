Setのすべての要素が同じenumの値である場合、SetはEnumSetに置き換えることができ、基礎となるデータ構造が単純なビットマップであるため、他のSetよりもはるかに効率的である。

コンプライアンス違反のコード例

public class MyClass {

  public enum COLOR {
    RED, GREEN, BLUE, ORANGE;
  }

  public void doSomething() {
    Set<COLOR> warm = new HashSet<COLOR>();
    warm.add(COLOR.RED);
    warm.add(COLOR.ORANGE);
  }
}

コンプライアンス・ソリューション

public class MyClass {

  public enum COLOR {
    RED, GREEN, BLUE, ORANGE;
  }

  public void doSomething() {
    Set<COLOR> warm = EnumSet.of(COLOR.RED, COLOR.ORANGE);
  }
}
