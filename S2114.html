コレクションをコレクション自身のメソッドの引数として渡すことは、他の引数を意図したエラーであるか、単に無意味なコードであるかのいずれかです。

さらに、メソッドによっては、実行中に引数が変更されないことが要求されるため、コレクションをそれ自身に渡すと、未定義の動作になることがあります。

コンプライアンス違反のコード例

List <Object> objs = new ArrayList<Object>();
objs.add("Hello");

objs.add(objs); // Noncompliant; StackOverflowException if objs.hashCode() called
objs.addAll(objs); // Noncompliant; behavior undefined
objs.containsAll(objs); // Noncompliant; always true
objs.removeAll(objs); // Noncompliant; confusing. Use clear() instead
objs.retainAll(objs); // Noncompliant; NOOP
