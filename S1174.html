Object.finalize() メソッドの契約は明確で、ガベージコレクタのみがこのメソッドを呼び出すことになっています。

このメソッドをpublicにすることは、呼び出し元が誰でも使えることを意味するため、誤解を招きやすい。

コンプライアンス違反のコード例

public class MyClass {

  @Override
  public void finalize() {    // Noncompliant
    /* ... */
  }
}
