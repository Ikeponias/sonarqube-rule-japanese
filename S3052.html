コンパイラは、クラス・フィールドに初期化値を設定する前に、自動的にデフォルト値に初期化するので、明示的にフィールドにデフォルト値を設定する必要はない。さらに、よりきれいなコードはより良いコードであるという論理のもと、そうすることは貧しいスタイルであると考えられています。

コンプライアンス違反のコード例

public class MyClass {

  int count = 0;  // Noncompliant
  // ...

}

コンプライアンス・ソリューション

public class MyClass {

  int count;
  // ...

}

例外あり

final フィールドは無視される。
