switch またはif チェーンのすべてのブランチが同じ実装であることはエラーです。コピーペーストの間違いで違うものを実行するか、switch/if チェーンが全く存在しないかのどちらかです。

コンプライアンス違反のコード例

if (b == 0) {  // Noncompliant
  doOneMoreThing();
} else {
  doOneMoreThing();
}

int b = a > 12 ? 4 : 4;  // Noncompliant

switch (i) {  // Noncompliant
  case 1:
    doSomething();
    break;
  case 2:
    doSomething();
    break;
  case 3:
    doSomething();
    break;
  default:
    doSomething();
}

例外あり

このルールは、else-s のないif チェーンや、default節のないswitch-es には適用されない。

if(b == 0) {    //no issue, this could have been done on purpose to make the code more readable
  doSomething();
} else if(b == 1) {
  doSomething();
}
