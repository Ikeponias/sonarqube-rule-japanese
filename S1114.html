Object.finalize() メソッドのオーバーライドは、一部のシステムリソースを破棄するため、慎重に行う必要があります。

このメソッド実装の最後にsuper.finalize() を呼び出すことは、親実装が何らかのシステムリソースを廃棄する必要がある場合に備えて、強く推奨されます。

コンプライアンス違反のコード例

protected void finalize() {   // Noncompliant; no call to super.finalize();
  releaseSomeResources();
}

protected void finalize() {
  super.finalize();  // Noncompliant; this call should come last
  releaseSomeResources();
}

コンプライアンス・ソリューション

protected void finalize() {
  releaseSomeResources();
  super.finalize();
}
