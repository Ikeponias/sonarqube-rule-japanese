コレクションのサイズと配列の長さは、常に0以上です。したがって、サイズや長さがゼロ以上であるかどうかをテストしても、結果は常にtrue であるため、意味がない。同様に、ゼロより小さいかどうかのテストは、常にfalse を返します。おそらく、コレクションや配列が空でないことをチェックするのが目的だったのでしょう。

コンプライアンス違反のコード例

if (myList.size() >= 0) { ... }

if (myList.size() < 0) { ... }

boolean result = myArray.length >= 0;

if (0 > myArray.length) { ... }

コンプライアンス・ソリューション

if (!myList.isEmpty()) { ... }

if (myArray.length >= 42) { ... }
