Java Language Specification-8.9による：

ネストしたenum型は暗黙のうちにstatic 。

だから、static 明示的に宣言する必要はない。

コンプライアンス違反のコード例

public class Flower {
  static enum Color { // Noncompliant; static is redundant here
    RED, YELLOW, BLUE, ORANGE
  }

  // ...
}

コンプライアンス・ソリューション

public class Flower {
  enum Color { // Compliant
    RED, YELLOW, BLUE, ORANGE
  }

  // ...
}
