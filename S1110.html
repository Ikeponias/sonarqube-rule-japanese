括弧の使用は、望ましい操作順序を強制する必要がない場合でも、コードの背後にある意図を明確にすることができます。しかし、冗長な括弧の組は誤解を招く可能性があるため、削除する必要があります。

コンプライアンス違反のコード例

int x = (y / 2 + 1);   //Compliant even if the parenthesis are ignored by the compiler

if (a && ((x+y > 0))) {  // Noncompliant
  //...
}

return ((x + 1));  // Noncompliant

コンプライアンス・ソリューション

int x = (y / 2 + 1);

if (a && (x+y > 0)) {
  //...
}

return (x + 1);
