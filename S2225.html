オブジェクトに対してtoString() やclone() を呼び出すと、常に文字列かオブジェクトが返されるはずです。代わりにnullを返すと、このメソッドの暗黙の契約に反することになります。

コンプライアンス違反のコード例

public String toString () {
  if (this.collection.isEmpty()) {
    return null; // Noncompliant
  } else {
    // ...

コンプライアンス・ソリューション

public String toString () {
  if (this.collection.isEmpty()) {
    return "";
  } else {
    // ...
