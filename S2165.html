ファイナライザでクラスフィールドをnull に設定することに意味はありません。もしこれがガベージコレクタへのヒントになるのであれば、それは不要です-オブジェクトはどのみちガベージコレクションされます-そして、そうすることは実際にガベージコレクタのために余分な仕事を引き起こすかもしれません。

コンプライアンス違反のコード例

public class Foo {
  private String name;

  @Override
  void finalize() {
    name = null;  // Noncompliant; completely unnecessary
  }
}
