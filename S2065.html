transient は、 クラスの中で、ファイル（またはストリーム）に書き出されないフィールドをマークするために使用されます。 を実装していないクラスでは、この修飾子は単にキー入力の無駄であり、削除する必要があります。Serializable Serializable

コンプライアンス違反のコード例

class Vegetable {  // does not implement Serializable
  private transient Season ripe;  // Noncompliant
  // ...
}

コンプライアンス・ソリューション

class Vegetable {
  private Season ripe;
  // ...
}
