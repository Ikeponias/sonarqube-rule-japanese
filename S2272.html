契約上、java.util.Iterator.next() メソッドの実装は、イテレーションの要素がなくなったときにNoSuchElementException 例外を投げる必要があります。イテレーションが終了したときの他の動作は、このIterator のユーザーにとって予期せぬ動作につながる可能性があります。

コンプライアンス違反のコード例

public class MyIterator implements Iterator<String>{
  ...
  public String next(){
    if(!hasNext()){
      return null;
    }
    ...
  }
}

コンプライアンス・ソリューション

public class MyIterator implements Iterator<String>{
  ...
  public String next(){
    if(!hasNext()){
      throw new NoSuchElementException();
    }
    ...
  }
}
