interface で宣言されたメソッドは、デフォルトでpublic とabstract になります。変数は自動的にpublic static final となります．最後に、class とinterface は自動的にpublic static となります。明示的に宣言する必要はありません．

アノテーションは暗黙のうちにインターフェースであるため、アノテーションも同様である。

同様に、final はfinal クラスのメソッド、private はEnum のコンストラクタ、static はclass やenum にネストされたinterface のメソッドで冗長となります。

コンプライアンス違反のコード例

public interface Vehicle {

  public void go(int speed, Direction direction);  // Noncompliant

コンプライアンス・ソリューション

public interface Vehicle {

  void go(int speed, Direction direction);
