1行で複数の変数を宣言するのは読みにくいです。

コンプライアンス違反のコード例

class MyClass {

  private int a, b;

  public void method(){
    int c; int d;
  }
}

コンプライアンス・ソリューション

class MyClass {

  private int a;
  private int b;

  public void method(){
    int c;
    int d;
  }
}

見る

  • CERT、DCL52-J。- 1回の宣言で2つ以上の変数を宣言しない
  • CERT、DCL04-C。- 1つの宣言で2つ以上の変数を宣言しない
