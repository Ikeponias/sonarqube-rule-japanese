部分式内の代入は発見しにくいので、コードの可読性が低下します。理想的には、部分式は副作用を持たないようにすることです。

コンプライアンス違反のコード例

if ((str = cont.substring(pos1, pos2)).isEmpty()) {  // Noncompliant
  //...

コンプライアンス・ソリューション

str = cont.substring(pos1, pos2);
if (str.isEmpty()) {
  //...

例外あり

while 文の条件中の代入、および関係式で囲まれた代入は無視される。

BufferedReader br = new BufferedReader(/* ... */);
String line;
while ((line = br.readLine()) != null) {...}
if ((i = j) >= 1) {...}

複合代入を含む連鎖代入は、無視されます。

int i = j = 0;
int k = (j += 1);
result = (bresult = new byte[len]);

見る

  • MITRE, CWE-481- 比較するのではなく、割り当てる。
  • CERT、EXP45-C。- 選択文での割り付けを行わない
  • CERT、EXP51-J。- 条件式で代入を実行しない
