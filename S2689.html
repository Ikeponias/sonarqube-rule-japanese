ObjectOutputStreamはシリアライズで使用され、ObjectOutputStream が最初に書き込むのはシリアライズストリームヘッダです。このヘッダーは1ファイルにつき1回、先頭に表示される必要があります。ObjectOutputStreamのコンストラクタにアペンドモードで開いたファイルを渡すと、ファイルの末尾にシリアライズストリームヘッダが追加され、その後にオブジェクトがアペンドされます。

ファイルからオブジェクトを読み返そうとすると、最初の1つだけが正常に読み込まれ、それ以降はStreamCorruptedException がスローされます。

コンプライアンス違反のコード例

FileOutputStream fos = new FileOutputStream (fileName , true);  // fos opened in append mode
ObjectOutputStream out = new ObjectOutputStream(fos);  // Noncompliant

コンプライアンス・ソリューション

FileOutputStream fos = new FileOutputStream (fileName);
ObjectOutputStream out = new ObjectOutputStream(fos);
