コーディング規約を共有することで、チームは効率的に共同作業を行うことができます。このルールでは、すべてのローカル変数、final 、初期化されたプリミティブ変数が、指定された正規表現に一致する名前を持っていることをチェックします。

コンプライアンス違反のコード例

デフォルトの正規表現で^[A-Z][A-Z0-9]*(_[A-Z0-9]+)*$ ：

public void doSomething() {
  final int local = 42;
  ...
}

コンプライアンス・ソリューション

public void doSomething() {
  final int LOCAL = 42;
  ...
}
