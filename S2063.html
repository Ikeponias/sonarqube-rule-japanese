シリアライズ不可能なComparator は、そうでなければSerializable で順序付けられたコレクションをシリアライズ不可能にすることができます。通常、Comparator をシリアライズ可能にするためのオーバヘッドは低いので、そうすることは良い防御的なプログラミングと考えることができます。

コンプライアンス違反のコード例

public class FruitComparator implements Comparator<Fruit> {  // Noncompliant
  int compare(Fruit f1, Fruit f2) {...}
  boolean equals(Object obj) {...}
}

コンプライアンス・ソリューション

public class FruitComparator implements Comparator<Fruit>, Serializable {
  private static final long serialVersionUID = 1;

  int compare(Fruit f1, Fruit f2) {...}
  boolean equals(Object obj) {...}
}
