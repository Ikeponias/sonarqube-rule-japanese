Files.exists メソッドは、JDK 8 では著しくパフォーマンスが低下し、実際に存在しないファイルをチェックするために使用すると、アプリケーションの速度が著しく低下することがあります。

java.nio.fileのパッケージからFiles.notExists 、Files.isDirectory 、Files.isRegularFile も同様です。

なお、このルールは、プロジェクトのsonar.java.source が8でない場合、自動的に無効になります。

コンプライアンス違反のコード例

Path myPath;
if(java.nio.file.Files.exists(myPath)) {  // Noncompliant
 // do something
}

コンプライアンス・ソリューション

Path myPath;
if(myPath.toFile().exists())) {
 // do something
}
