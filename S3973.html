中括弧で囲まれていない場合、条件付きの直後の行が条件付きで実行される行となります。慣例的にも、良い習慣としても、このような行はインデントされます。中括弧と字下げの両方がない場合、元のプログラマーの意図はまったく不明であり、おそらく実際に実行されるものではありません。さらに、このようなコードは、保守担当者を混乱させる可能性が高い。

コンプライアンス違反のコード例

if (condition)  // Noncompliant
doTheThing();

doTheOtherThing();
somethingElseEntirely();

foo();

コンプライアンス・ソリューション

if (condition)
  doTheThing();

doTheOtherThing();
somethingElseEntirely();

foo();
