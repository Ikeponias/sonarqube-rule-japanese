return やcontinue などのジャンプ文は、プログラムの実行の既定の流れを変えることができますが、制御の流れを元の方向に向けるジャンプ文は、キー入力の無駄遣いにしかなりません。

コンプライアンス違反のコード例

public void foo() {
  while (condition1) {
    if (condition2) {
      continue; // Noncompliant
    } else {
      doTheThing();
    }
  }
  return; // Noncompliant; this is a void method
}

コンプライアンス・ソリューション

public void foo() {
  while (condition1) {
    if (!condition2) {
      doTheThing();
    }
  }
}
