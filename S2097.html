equals メソッドは、一般的なObject をパラメータとして受け取るため、どのようなタイプのオブジェクトでも渡すことができます。メソッドは、そのクラスの型のオブジェクトをテストするためにのみ使用されると仮定してはいけません。その代わりに、パラメータの型をチェックする必要があります。

コンプライアンス違反のコード例

public boolean equals(Object obj) { // Noncompliant
  MyClass mc = (MyClass)obj;
  // ...
}

コンプライアンス・ソリューション

public boolean equals(Object obj) {
  if (obj == null)
    return false;

  if (this.getClass() != obj.getClass())
    return false;

  MyClass mc = (MyClass)obj;
  // ...
}
