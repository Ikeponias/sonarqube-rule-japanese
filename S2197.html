負の数のモジュラスを計算すると、結果は負かゼロのどちらかになります。したがって、等式を求める変数のモジュラスを正の数（または負の数）と比較すると、予期せぬ結果になることがあります。

コンプライアンス違反のコード例

public boolean isOdd(int x) {
  return x % 2 == 1;  // Noncompliant; if x is an odd negative, x % 2 == -1
}

コンプライアンス・ソリューション

public boolean isOdd(int x) {
  return x % 2 != 0;
}
