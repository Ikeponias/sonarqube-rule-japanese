Oracle社が定義したJava Code Conventionsによると、クラスまたはインターフェイス宣言のメンバーは、ソースファイル内で以下の順序で表示される必要があります：

  • クラス変数
  • インスタンス変数
  • コンストラクタ
  • メソッド

コンプライアンス違反のコード例

public class Foo{
   private int field = 0;
   public boolean isTrue() {...}
   public Foo() {...}                         // Noncompliant, constructor defined after methods
   public static final int OPEN = 4;  //Noncompliant, variable defined after constructors and methods
}

コンプライアンス・ソリューション

public class Foo{
   public static final int OPEN = 4;
   private int field = 0;
   public Foo() {...}
   public boolean isTrue() {...}
}
