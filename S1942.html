Javaのimport メカニズムでは、単純なクラス名を使用することができます。したがって、importのファイル内でクラスの完全修飾名を使用することは、冗長で混乱を招くことになります。

コンプライアンス違反のコード例

import java.util.List;
import java.sql.Timestamp;

//...

java.util.List<String> myList;  // Noncompliant
java.sql.Timestamp tStamp; // Noncompliant

コンプライアンス・ソリューション

import java.util.List;
import java.sql.Timestamp;

//...

List<String> myList;
Timestamp tStamp;
