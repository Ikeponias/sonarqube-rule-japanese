static,private 以外のメンバーを持つ非抽象クラスおよび列挙型は、コンストラクタ内またはデフォルト値で、これらのメンバーを明示的に初期化する必要があります。

コンプライアンス違反のコード例

class A { // Noncompliant
  private int field;
}

コンプライアンス・ソリューション

class A {
  private int field;

  A(int field) {
    this.field = field;
  }
}

例外あり

  • Builder パターンを実装したクラス（名称が "Builder" で終わるもの）。
  • でアノテーションされたJava EEクラス：
    • ManagedBean
    • メッセージドリブン
    • シングルトン
    • ステートフル
    • ステートレス
    • ウェブサービス
    • ウェブフィルタ
    • ウェブサーブレット
  • クラスとフィールドにアノテーションを施した
    • プレクサス コンポーネントアノテーション
    • メイブンモジョ
  • フィールドに注釈をつける：
    • リソース
    • イージェービー
    • インジェクト
    • オートワイヤード
