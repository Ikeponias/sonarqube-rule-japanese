Collection.size() を使って空虚さをテストするのはうまくいくが、Collection.isEmpty() を使うとコードが読みやすくなり、より高性能になる可能性がある。isEmpty() メソッドの実装の時間的複雑さはO(1) であるべきですが、size() の実装の中にはO(n) になるものもあります。

コンプライアンス違反のコード例

if (myCollection.size() == 0) {  // Noncompliant
  /* ... */
}

コンプライアンス・ソリューション

if (myCollection.isEmpty()) {
  /* ... */
}
