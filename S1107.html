コーディング規約を共有することで、チームでの効率的なコラボレーションが可能になります。

このルールにより、閉じ中括弧を次のelse,catch,finally キーワードと同じ行に置くことが必須となります。

コンプライアンス違反のコード例

public void myMethod() {
  if(something) {
    executeTask();
  } else if (somethingElse) {
    doSomethingElse();
  }
  else {                               // Noncompliant
     generateError();
  }

  try {
    generateOrder();
  } catch (Exception e) {
    log(e);
  }
  finally {                            // Noncompliant
    closeConnection();
  }
}

コンプライアンス・ソリューション

public void myMethod() {
  if(something) {
    executeTask();
  } else if (somethingElse) {
    doSomethingElse();
  } else {
     generateError();
  }

  try {
    generateOrder();
  } catch (Exception e) {
    log(e);
  } finally {
    closeConnection();
  }
}
