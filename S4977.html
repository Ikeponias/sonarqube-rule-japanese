シャドーイングを行うと、外側スコープから型パラメータを使用することができなくなります。また、どの型パラメータが使用されているのか区別がつかなくなることもあります。

このルールでは、内側スコープの型パラメーターが外側スコープの型パラメーターと同じ名前を使用している場合に問題が発生します。

コンプライアンス違反のコード例

 public class TypeParameterHidesAnotherType<T> {

    public class Inner<T> { // Noncompliant
      //...
    }

    private <T> T method() { // Noncompliant
      return null;
    }

  }

コンプライアンス・ソリューション

public class NoTypeParameterHiding<T> {

    public class Inner<S> { // Compliant
      List<S> listOfS;
    }

    private <V> V method() { // Compliant
      return null;
    }

  }
