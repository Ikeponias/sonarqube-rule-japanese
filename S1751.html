最大1回の反復を伴うループは、if ステートメントを使用して1つのコードを条件付きで実行することと同じです。開発者は、このようなループ文の使い方を見つけることを期待していません。もし作者の最初の意図が本当に1つのコードを条件付きで実行することであったなら、代わりにif ステートメントを使用すべきです。

最悪の場合、それは著者の最初の意図ではなかったので、ループの本体は、より適切な方法で入れ子のreturn,break またはthrow 文を使用するように修正する必要があります。

コンプライアンス違反のコード例

for (int i = 0; i < 10; i++) { // noncompliant, loop only executes once
  printf("i is %d", i);
  break;
}
...
for (int i = 0; i < 10; i++) { // noncompliant, loop only executes once
  if (i == x) {
    break;
  } else {
    printf("i is %d", i);
    return;
  }
}

コンプライアンス・ソリューション

for (int i = 0; i < 10; i++) {
  printf("i is %d", i);
}
...
for (int i = 0; i < 10; i++) {
  if (i == x) {
    break;
  } else {
    printf("i is %d", i);
  }
}
