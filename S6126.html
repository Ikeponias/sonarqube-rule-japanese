Java 15ではText Blocksが正式採用され、使用できるようになりました。Java < 15で複数行の文字列を記述する場合、String concatenationを記述するのが最も一般的なパターンでした。それがText Blocksを使うことでより自然な形で行えるようになりました。

コンプライアンス違反のコード例

String textBlock =
               "<html>\n" +
               "    <body>\n" +
               "        <tag>\n" +
               "        </tag>\n" +
               "    </body>\n" +
               "</html>";

コンプライアンス・ソリューション

String textBlock = """
        <html>
            <body>
                <tag>
                </tag>
            </body>
        </html>""";
