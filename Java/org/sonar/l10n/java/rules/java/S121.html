技術的には間違っていませんが、中括弧の省略は誤解を招く恐れがあり、メンテナンス時にエラーが発生する可能性があります。

コンプライアンス違反のコード例

if (condition)  // Noncompliant
  executeSomething();

コンプライアンス・ソリューション

if (condition) {
  executeSomething();
}

例外あり

if ステートメントの本文が、単一のreturn 、break またはcontinue であり、同一行にある場合。

見る

  • CERT、EXP19-C。- if文、for文、while文の本文に中括弧を使用する。
  • CERT、EXP52-J。- if文、for文、while文の本文に中括弧を使用する。
