サブパターンを括弧で囲んでグループを作ることができる。これにより、交替を制限したり、グループを後方参照したり、サブパターンに量化子を適用することができる。

このグループが試合結果の一部であってはならない場合、またはこのグループへの参照が不要な場合は、冒頭の括弧の後ろに?: を追加することで、非捕捉グループを作成することができます。

しかし、この非捕捉グループが量化子を持たない場合、あるいは交替を包まない場合、このグループの画像化は冗長となる。

コンプライアンス違反のコード例

"(?:number)\\d{2}"

コンプライアンス・ソリューション

"number\\d{2}"      	// it is anyway required
"(?:number)?\\d{2}"  	// it is in fact optional

例外あり

本ルールでは、非捕獲グループが交代制である場合、問題は報告されない。

"(?:number|string)"
