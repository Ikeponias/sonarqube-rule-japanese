interface のメソッドと同じシグネチャを持つabstract のメソッドを、implements というクラスで冗長に定義する意味はないでしょう。具体的な子クラスは、いずれにせよそのメソッドを実装しなければなりません。

コンプライアンス違反のコード例

public interface Reportable {
  String getReport();
}

public abstract class AbstractRuleReport implements Reportable{
  public abstract String getReport();  // Noncompliant

  // ...
}
