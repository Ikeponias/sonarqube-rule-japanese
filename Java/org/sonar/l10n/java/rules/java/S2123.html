インクリメントまたはデクリメントされた後、保存されない値は、良く言えば無駄なコード、悪く言えばバグです。

コンプライアンス違反のコード例

public int pickNumber() {
  int i = 0;
  int j = 0;

  i = i++; // Noncompliant; i is still zero

  return j++; // Noncompliant; 0 returned
}

コンプライアンス・ソリューション

public int pickNumber() {
  int i = 0;
  int j = 0;

  i++;
  return ++j;
}
