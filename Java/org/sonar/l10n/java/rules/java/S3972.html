コードは、各ステートメントが独立した行を持つとき、最も明確になります。それでも、if とその結果のthen文を同じ行にまとめるのは、よくあるパターンです。しかし、if が、直前のthen、else、else if部分の終了} と同じ行に置かれた場合、それはエラーになるか、else が欠けているか、あるいは、2つの文がつながっていないことを保守者が理解できず、将来のエラーにつながるかのどちらかです。

コンプライアンス違反のコード例

if (condition1) {
  // ...
} if (condition2) {  // Noncompliant
  //...
}

コンプライアンス・ソリューション

if (condition1) {
  // ...
} else if (condition2) {
  //...
}

または

if (condition1) {
  // ...
}

if (condition2) {
  //...
}
