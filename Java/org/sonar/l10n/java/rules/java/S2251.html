カウンタが間違った方向に動くfor ループは、無限ループではありません。ラップアラウンドのため、ループは最終的に停止条件に達しますが、その際、予想よりも何度も何度も実行され、予期せぬ動作を引き起こす可能性があります。

コンプライアンス違反のコード例

public void doSomething(String [] strings) {
  for (int i = 0; i < strings.length; i--) { // Noncompliant;
    String string = strings[i];  // ArrayIndexOutOfBoundsException when i reaches -1
    //...
  }

コンプライアンス・ソリューション

public void doSomething(String [] strings) {
  for (int i = 0; i < strings.length; i++) {
    String string = strings[i];
    //...
  }
