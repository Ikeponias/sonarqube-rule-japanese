java.util.concurrent.ScheduledThreadPoolExecutorのプールはcorePoolSize でサイズ調整されるので、corePoolSizeをゼロに設定すると、実行者はスレッドを持たず、何も実行しないことになる。

このルールは、corePoolSize がセッターまたはオブジェクトコンストラクタによってゼロに設定されるインスタンスを検出します。

コンプライアンス違反のコード例

public void do(){

  stpe2.setCorePoolSize(0);  // Noncompliant
