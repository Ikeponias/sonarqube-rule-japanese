引数がない場合、Collections.toArray メソッドはObject [] を返します。これを適切なクラスの配列にキャストしようとすると、実行時にClassCastException が発生します。代わりに、正しい型の配列を呼び出しに渡します。

コンプライアンス違反のコード例

public String [] getStringArray(List<String> strings) {
  return (String []) strings.toArray();  // Noncompliant; ClassCastException thrown
}

コンプライアンス・ソリューション

public String [] getStringArray(List<String> strings) {
  return strings.toArray(new String[0]);
}
