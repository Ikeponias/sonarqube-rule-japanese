ダブルブレース初期化（DBI）は、所有するオブジェクトのインスタンスへの参照を持つ匿名クラスを作成するため、匿名内部クラスが他のオブジェクトによって返され保持される場合、その使用はメモリリークにつながる可能性があります。リークがない場合でも、DBIは非常にわかりにくいので、ほとんどの保守者が混乱することになります。

コレクションの場合は、代わりにArrays.asList を使用するか、明示的に各アイテムをコレクションに直接追加します。

コンプライアンス違反のコード例

Map source = new HashMap(){{ // Noncompliant
    put("firstName", "John");
    put("lastName", "Smith");
}};

コンプライアンス・ソリューション

Map source = new HashMap();
// ...
source.put("firstName", "John");
source.put("lastName", "Smith");
// ...
