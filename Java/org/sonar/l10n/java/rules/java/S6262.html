AWSクライアントで明示的にリージョンを設定する場合、ハードコードされたStringではなく、EnumRegionsから値を提供することを常に好むべきです。 これにより、APIの変更に透過的に対応でき、間違いを避けることができます。

このルールでは、利用可能な列挙値の代わりにハードコードされた文字列が使用される場合の問題を報告します。

コンプライアンス違反のコード例

AmazonS3ClientBuilder.standard().withRegion("eu_west_1").build();

コンプライアンス・ソリューション

AmazonS3ClientBuilder.standard().withRegion(Regions.EU_WEST_1).build();
