コードをわかりやすくするために、base クラスのstatic メンバには、派生型の名前を使用してアクセスしてはいけません。そうすると混乱し、2 つの異なる静的メンバが存在するように錯覚してしまう可能性があります。

コンプライアンス違反のコード例

class Parent {
  public static int counter;
}

class Child extends Parent {
  public Child() {
    Child.counter++;  // Noncompliant
  }
}

コンプライアンス・ソリューション

class Parent {
  public static int counter;
}

class Child extends Parent {
  public Child() {
    Parent.counter++;
  }
}
