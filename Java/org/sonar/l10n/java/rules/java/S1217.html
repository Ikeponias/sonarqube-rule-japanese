Thread.run() メソッドの目的は、コードを別の専用スレッドで実行することです。このメソッドを直接呼び出すと、そのコードが現在のスレッドで実行されてしまうので、意味がありません。

期待通りの動作を得るには、代わりにThread.start() メソッドを呼び出してください。

コンプライアンス違反のコード例

Thread myThread = new Thread(runnable);
myThread.run(); // Noncompliant

コンプライアンス・ソリューション

Thread myThread = new Thread(runnable);
myThread.start(); // Compliant
