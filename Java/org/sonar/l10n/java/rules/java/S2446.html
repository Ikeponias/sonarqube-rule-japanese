notify と はどちらも眠っているスレッドを起こしますが、 は1つしか起こさず、 はすべてのスレッドを起こします。 は正しいスレッドを起こさないかもしれないので、 を代わりに使用する必要があります。notifyAll notify notifyAll notify notifyAll

コンプライアンス違反のコード例

class MyThread extends Thread{

  @Override
  public void run(){
    synchronized(this){
      // ...
      notify();  // Noncompliant
    }
  }
}

コンプライアンス・ソリューション

class MyThread extends Thread{

  @Override
  public void run(){
    synchronized(this){
      // ...
      notifyAll();
    }
  }
}
