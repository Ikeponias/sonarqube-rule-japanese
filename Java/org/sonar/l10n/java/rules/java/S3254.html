アノテーションパラメータのデフォルト値の指定は冗長である。読みやすさを考慮し、このような値は省略されるべきである。

コンプライアンス違反のコード例

@MyAnnotation(arg = "def")  // Noncompliant
public class MyClass {
  // ...
}
public @interface MyAnnotation {
  String arg() default "def";
}

コンプライアンス・ソリューション

@MyAnnotation
public class MyClass {
  // ...
}
public @interface MyAnnotation {
  String arg() default "def";
}
