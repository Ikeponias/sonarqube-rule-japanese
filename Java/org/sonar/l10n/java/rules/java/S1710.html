Java 8以前は、同じアノテーションの複数のインスタンスを使用する必要がある場合、それらをコンテナアノテーションでラップする必要がありました。Java 8では、その必要がなくなり、よりきれいで読みやすいコードを実現できるようになりました。

なお、このルールは、プロジェクトのsonar.java.source が8 よりも低い場合、自動的に無効となります。

コンプライアンス違反のコード例

@SomeAnnotations({  // Noncompliant
  @SomeAnnotation(..a..),
  @SomeAnnotation(..b..),
  @SomeAnnotation(..c..),
})
public class SomeClass {
  ...
}

コンプライアンス・ソリューション

@SomeAnnotation(..a..)
@SomeAnnotation(..b..)
@SomeAnnotation(..c..)
public class SomeClass {
  ...
}
