ラベルが宣言されていても、プログラム中で使用されない場合は、デッドコードとみなされるため、削除する必要があります。

これにより、開発者が「このラベルは何に使うんだろう」と迷うことがなくなり、保守性が向上します。

コンプライアンス違反のコード例

void foo() {
  outer: //label is not used.
  for(int i = 0; i<10; i++) {
    break;
  }
}

コンプライアンス・ソリューション

void foo() {
  for(int i = 0; i<10; i++) {
    break;
  }
}

  • CERT、MSC12-C。- 効果がない、または実行されないコードを検出して削除する
