JUnitTestCase にテストメソッドがないのでは意味がありません。同様に、tests ディレクトリに*Test 、*Tests 、*TestCase という名前のファイルがありながら、その中にテストがないのもいけません。このようなことをすると、カバーされていないクラスがテストされたと思われる可能性があります。

このルールは、testディレクトリ内のファイルの名前が*Test 、*Tests 、または*TestCase 、またはTestCase を実装しているが、テストを含んでいない場合に問題を発生させます。

対応するフレームワーク

  • ジュウニットスリー
  • ジュウニットフォー
  • ジュウニットファイブ
  • テストエヌジー
  • ゾハク
  • アーキユニット
  • 盟約
