テキストブロックのインデントには、スペースのみ、またはタブのみを使用します。空白が混ざると、インデントが不規則な結果になります。

コンプライアンス違反のコード例

String textBlock = """
        this is
<tab>text block!
        !!!!
      """;

コンプライアンス・ソリューション

String textBlock = """
        this is
        text block!
        !!!!
      """;
