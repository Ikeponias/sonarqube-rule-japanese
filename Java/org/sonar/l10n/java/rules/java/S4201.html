instanceof のテストと合わせてヌルテストをする必要はありません。null はinstanceof の何かではありませんから、ヌルチェックは冗長です。

コンプライアンス違反のコード例

if (x != null && x instanceof MyClass) { ... }  // Noncompliant

if (x == null || ! x instanceof MyClass) { ... } // Noncompliant

コンプライアンス・ソリューション

if (x instanceof MyClass) { ... }

if (! x instanceof MyClass) { ... }
