FIXME タグは、バグが疑われるが、開発者が後で対処したい場所をマークするためによく使われます。

開発者に時間がなかったり、そのタグに戻ることを忘れてしまったりすることもあります。

このルールは、それらのタグを追跡し、気づかれないようにするためのものです。

コンプライアンス違反のコード例

int divide(int numerator, int denominator) {
  return numerator / denominator;              // FIXME denominator value might be  0
}
