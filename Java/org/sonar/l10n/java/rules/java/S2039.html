メンバ変数の可視性を明示的に宣言しないと、予想外の可視性を持つことになり、他のクラスから予期せぬ改変を受ける可能性があります。

コンプライアンス違反のコード例

class Ball {
    String color="red";  // Noncompliant
}
enum A {
  B;
  int a;
}

コンプライアンス・ソリューション

class Ball {
    private String color="red";  // Compliant
}
enum A {
  B;
  private int a;
}

例外あり

@VisibleForTesting のアノテーションが付いたメンバーは無視されます。これは、コードをテスト可能にするために、意図的に可視性を緩和したことを示すからです。

class Cone {
  @VisibleForTesting
  Logger logger; // Compliant
}

非推奨

このルールは非推奨であり、いずれ削除される予定です。
