コーディング規約を共有することで、チームでの効率的なコラボレーションを可能にします。

このルールにより、閉じ中括弧と次のelse,catch またはfinally キーワードを異なる2行に配置することが必須となります。

コンプライアンス違反のコード例

public void myMethod() {
  if(something) {
    executeTask();
  } else if (somethingElse) {          // Noncompliant
    doSomethingElse();
  }
  else {                               // Compliant
     generateError();
  }

  try {
    generateOrder();
  } catch (Exception e) {
    log(e);
  }
  finally {
    closeConnection();
  }
}

コンプライアンス・ソリューション

public void myMethod() {
  if(something) {
    executeTask();
  }
  else if (somethingElse) {
    doSomethingElse();
  }
  else {
     generateError();
  }

  try {
    generateOrder();
  }
  catch (Exception e) {
    log(e);
  }
  finally {
    closeConnection();
  }
}
