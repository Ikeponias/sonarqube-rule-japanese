数字に含まれる小グループを区切るために句読点を使用すると、数字をより読みやすくすることができます。例えば、1,000,000,000と1000000000を比較してみましょう。しかし、1,000,00,000のようにグループ分けが不規則な場合は、エラーとなります。

このルールは、アンダースコア（_ ）を使って数字を不規則な小グループに分割する場合に問題となる。

コンプライアンス違反のコード例

int duos = 1_00_00;
int million = 1_000_00_000;  // Noncompliant
int thousand = 1000;
int tenThousand = 100_00;  // Noncompliant
