コンストラクタでstatic フィールドに値を割り当てると、クラスのすべてのインスタンスで値が変更されるため、実行時に信頼できない動作が発生する可能性があります。

その代わりに、フィールドのstatic モディファイアを削除するか、静的に初期化します。

コンプライアンス違反のコード例

public class Person {
  static Date dateOfBirth;
  static int expectedFingers;

  public Person(date birthday) {
    dateOfBirth = birthday;  // Noncompliant; now everyone has this birthday
    expectedFingers = 10;  // Noncompliant
  }
}

コンプライアンス・ソリューション

public class Person {
  Date dateOfBirth;
  static int expectedFingers = 10;

  public Person(date birthday) {
    dateOfBirth = birthday;
  }
}
