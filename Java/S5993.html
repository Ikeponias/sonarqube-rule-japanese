抽象クラスは、パブリックコンストラクタを持つべきではありません。抽象クラスのコンストラクタは、そのサブクラスのコンストラクタの中でしか呼び出すことができません。ですから、コンストラクタをパブリックにする意味はありません。protected 修飾子で十分でしょう。

コンプライアンス違反のコード例

public abstract class AbstractClass1 {
    public AbstractClass1 () { // Noncompliant, has public modifier
        // do something here
    }
}

コンプライアンス・ソリューション

public abstract class AbstractClass2 {
    protected AbstractClass2 () {
        // do something here
    }
}
