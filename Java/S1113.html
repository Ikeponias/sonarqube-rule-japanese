Object.finalize() メソッドは、ガベージコレクタがオブジェクトへの参照がもうないと判断したときに、オブジェクトに対して呼び出されます。しかし、このメソッドが、オブジェクトへの最後の参照が削除されると同時に呼び出されることは全く保証されていません。数マイクロ秒から数分後に呼ばれることもあります。ですから、オブジェクトによってシステムリソースを廃棄する必要がある場合、この非同期メカニズムに頼って廃棄しないほうがよいでしょう。

コンプライアンス違反のコード例

public class MyClass {
  ...
  protected void finalize() {
    releaseSomeResources();    // Noncompliant
  }
  ...
}
