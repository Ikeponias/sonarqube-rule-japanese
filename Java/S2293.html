Java 7では、ジェネリックスコードの冗長性を軽減するために、ダイヤモンド演算子(<>)が導入されました。例えば、List'の型は、その宣言とコンストラクタの両方で宣言する必要がありましたが、コンストラクタの宣言を<> で簡略化できるようになり、コンパイラがその型を推論するようになりました。

なお、このルールは、プロジェクトのsonar.java.source が7 よりも低い場合、自動的に無効となります。

コンプライアンス違反のコード例

List<String> strings = new ArrayList<String>();  // Noncompliant
Map<String,List<Integer>> map = new HashMap<String,List<Integer>>();  // Noncompliant

コンプライアンス・ソリューション

List<String> strings = new ArrayList<>();
Map<String,List<Integer>> map = new HashMap<>();
