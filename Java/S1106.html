コーディング規約を共有することで、効率的なコラボレーションが可能になります。この規則では、開いた中括弧を行頭に置くことが義務付けられています。

コンプライアンス違反のコード例

public void myMethod {  // Noncompliant
  if(something) {  // Noncompliant
    executeTask();
  } else {  // Noncompliant
    doSomethingElse();
  }
}

コンプライアンス・ソリューション

public void myMethod
{
  if(something)
  {
    executeTask();
  } else
  {
    doSomethingElse();
  }
}
