ゼロで始まる整数リテラルは、10進数ではなく8進数です。8進数の使用は完全にサポートされていますが、ほとんどの開発者は8進数を使用した経験がありません。そのため、8進数を10進数として認識せず、10進数と勘違いしてしまうことがあります。

コンプライアンス違反のコード例

int myNumber = 010; // Noncompliant. myNumber will hold 8, not 10 - was this really expected?

コンプライアンス・ソリューション

int myNumber = 8;

見る

  • CERT、DCL18-C。- 10進数指定時に整数定数を0から始めない
  • CERT、DCL50-J。- 視覚的に区別できる識別子を使用する
