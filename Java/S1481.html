ローカル変数が宣言されているのに使われていない場合、それはデッドコードであり、削除されるべきです。そうすることで、開発者がその変数が何に使われているのか疑問に思うことがなくなるので、保守性が向上します。

コンプライアンス違反のコード例

public int numberOfMinutes(int hours) {
  int seconds = 0;   // seconds is never used
  return hours * 60;
}

コンプライアンス・ソリューション

public int numberOfMinutes(int hours) {
  return hours * 60;
}
