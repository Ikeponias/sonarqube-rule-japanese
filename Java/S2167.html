重要なのは、compareTo から返される値の大きさではなく、符号である。Integer.MIN_VALUEを返しても、より高度な不等式を伝えることはできません。また、compareTo の返り値が、負の値が正になることを期待して反転されることがあるので、そうするとエラーが発生することがあります。しかし，Integer.MIN_VALUE を反転させると，Integer.MAX_VALUE ではなく，Integer.MIN_VALUE が返されます．

コンプライアンス違反のコード例

public int compareTo(MyClass) {
  if (condition) {
    return Integer.MIN_VALUE;  // Noncompliant
  }

コンプライアンス・ソリューション

public int compareTo(MyClass) {
  if (condition) {
    return -1;
  }
