メッセージをログに記録する場合、いくつかの重要な要件を満たす必要があります：

  • ユーザーが簡単にログを取り出すことができること
  • ログに記録されるメッセージの形式は、ユーザーが簡単に読めるように統一されていなければならない
  • 記録されたデータは、実際に記録されている必要がある
  • 機密性の高いデータは、安全にログを取る必要があります

プログラムが標準出力に直接書き込む場合、これらの要件に準拠する方法は全くありません。そのため、専用のロガーを定義して使用することが強く推奨されます。

コンプライアンス違反のコード例

System.out.println("My Message");  // Noncompliant

コンプライアンス・ソリューション

logger.log("My Message");
