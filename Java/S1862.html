if/else if の連鎖は、上から下へ評価される。実行されるのは、最大でも1つの分岐だけです。true と評価される条件を持つ最初の分岐が実行されます。

したがって、条件を重複させると自動的にデッドコードになります。通常、これはコピー＆ペーストのミスによるものです。よく言えば単なるデッドコード、悪く言えばバグであり、コードをメンテナンスしていくうちにさらなるバグを誘発する可能性が高く、明らかに予期せぬ挙動につながる可能性があります。

コンプライアンス違反のコード例

if (param == 1)
  openWindow();
else if (param == 2)
  closeWindow();
else if (param == 1)  // Noncompliant
  moveWindowToTheBackground();
}

コンプライアンス・ソリューション

if (param == 1)
  openWindow();
else if (param == 2)
  closeWindow();
else if (param == 3)
  moveWindowToTheBackground();
}
