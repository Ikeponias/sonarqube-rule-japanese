switch ステートメントは、同じ式の値によって異なるケースが多数存在する場合に便利です。

しかし、たった1つか2つのケースについては、if ステートメントを使用することで、より読みやすいコードになります。

コンプライアンス違反のコード例

switch (variable) {
  case 0:
    doSomething();
    break;
  default:
    doSomethingElse();
    break;
}

コンプライアンス・ソリューション

if (variable == 0) {
  doSomething();
} else {
  doSomethingElse();
}
