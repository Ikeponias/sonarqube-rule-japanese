リストや数値の集合など、同じデータを複数回ループするメソッドがある場合、ループを1つの反復処理にまとめることで、より効率的なメソッドになる可能性が高い。

コンプライアンス違反のコード例

public void doSomethingToAList(List<String> strings) {
  for (String str : strings) {
    doStep1(str);
  }
  for (String str : strings) {  // Noncompliant
    doStep2(str);
  }
}

コンプライアンス・ソリューション

public void doSomethingToAList(List<String> strings) {
  for (String str : strings) {
    doStep1(str);
    doStep2(str);
  }
}
