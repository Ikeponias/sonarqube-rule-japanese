公式のjavadocドキュメントによると、このObject.finalize()は、ガベージコレクションがオブジェクトへの参照がもうないと判断したときに、オブジェクトに対してガベージコレクタが呼び出すものです。このメソッドを明示的に呼び出すと、この契約を破ることになるので、誤解を招きやすい。

コンプライアンス違反のコード例

public void dispose() throws Throwable {
  this.finalize();                       // Noncompliant
}
