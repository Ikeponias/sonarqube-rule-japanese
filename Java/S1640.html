Mapのすべてのキーが同じenumからの値である場合、Map は、EnumMap に置き換えることができます。これは、基礎となるデータ構造が単純な配列であるため、他のセットよりもはるかに効率的です。

コンプライアンス違反のコード例

public class MyClass {

  public enum COLOR {
    RED, GREEN, BLUE, ORANGE;
  }

  public void mapMood() {
    Map<COLOR, String> moodMap = new HashMap<COLOR, String> ();
  }
}

コンプライアンス・ソリューション

public class MyClass {

  public enum COLOR {
    RED, GREEN, BLUE, ORANGE;
  }

  public void mapMood() {
    EnumMap<COLOR, String> moodMap = new EnumMap<> (COLOR.class);
  }
}
