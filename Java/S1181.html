Throwable Error は、Javaのすべてのエラーと例外のスーパークラスです。アプリケーションによって捕捉されることを意図していない、すべてのエラーのスーパークラスです。

Throwable とError のいずれかをキャッチすると、OutOfMemoryError とInternalError もキャッチすることになり、アプリケーションはそこから回復しようとしない。

コンプライアンス違反のコード例

try { /* ... */ } catch (Throwable t) { /* ... */ }
try { /* ... */ } catch (Error e) { /* ... */ }

コンプライアンス・ソリューション

try { /* ... */ } catch (RuntimeException e) { /* ... */ }
try { /* ... */ } catch (MyException e) { /* ... */ }
