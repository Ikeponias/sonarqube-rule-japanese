単一のステートメントを含むラムダの書き方には2つの方法がありますが、1つの方がコンパクトで読みやすいのは確かです。

なお、このルールは、プロジェクトのsonar.java.source が8 よりも低い場合、自動的に無効となります。

コンプライアンス違反のコード例

x -> {System.out.println(x+1);}
(a, b) -> { return a+b; }

コンプライアンス・ソリューション

x -> System.out.println(x+1)
(a, b) -> a+b    //For return statement, the return keyword should also be dropped
