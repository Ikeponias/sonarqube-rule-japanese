アサーションでリテラルなブーリアン値やヌルを使用する理由はありません。assertEquals やassertNotEqualsなどのメソッドでこれらを使うのではなく、assertTrue やassertFalse、assertNullやassertNotNullを使うべきでしょう (Fest を使う場合はisNullなど)。assertNullのような等式とは関係のないアサーションでこれらを使用することは、ほとんどの場合バグである。

対応するフレームワーク

  • ジュウニットスリー
  • ジュウニットフォー
  • ジュウニットファイブ
  • フェストアサート

コンプライアンス違反のコード例

Assert.assertTrue(true);  // Noncompliant
assertThat(null).isNull(); // Noncompliant

assertEquals(true, something()); // Noncompliant
assertNotEquals(null, something()); // Noncompliant

コンプライアンス・ソリューション

assertTrue(something());
assertNotNull(something());
