Javaの進化の過程で、キーワードが追加されました。これらの単語を識別子として使用するコードは、古いバージョンのJavaではコンパイル可能ですが、最新のバージョンではコンパイルできません。

以下のキーワードは、無効な識別子としてマークされます。

キーワード 追加

_

9

enum

5.0

assert や も有効な識別子の一例で、後のバージョンでキーワードとなったが、このルールではサポートされない。strictfp

コンプライアンス違反のコード例

public void doSomething() {
  int enum = 42;            // Noncompliant
  String _ = "";   // Noncompliant
}

コンプライアンス・ソリューション

public void doSomething() {
  int magic = 42;
}
