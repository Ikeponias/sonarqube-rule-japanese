ストリーム操作には、別のストリームを返す中間操作と、ストリーム以外のものを返す終端操作の2種類があります。中間処理は遅延処理であり、その結果に対して終端処理が行われない限り、実際には実行されない。従って、中間ストリーム操作の結果がターミナル操作に供給されない場合、それは何の役にも立たず、ほぼ間違いなくエラーとなる。

コンプライアンス違反のコード例

widgets.stream().filter(b -> b.getColor() == RED); // Noncompliant

コンプライアンス・ソリューション

int sum = widgets.stream()
                      .filter(b -> b.getColor() == RED)
                      .mapToInt(b -> b.getWeight())
                      .sum();
Stream<Widget> pipeline = widgets.stream()
                                 .filter(b -> b.getColor() == GREEN)
                                 .mapToInt(b -> b.getWeight());
sum = pipeline.sum();
