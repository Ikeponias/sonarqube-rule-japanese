コンストラクタでThread.start() を呼び出すことの問題点は、クラスが拡張された場合、混乱が生じることです。

このルールは、final 以外のクラスのコンストラクタでstart が呼び出された場合に、問題を生じさせます。

コンプライアンス違反のコード例

public class MyClass {

  Thread thread = null;

  public MyClass(Runnable runnable) {
    thread = new Thread(runnable);
    thread.start(); // Noncompliant
  }
}
