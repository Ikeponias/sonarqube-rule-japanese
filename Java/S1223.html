クラスとそのメソッドの一部が同じ名前を持つことは誤解を招き、意図的にそうしたのか、それともそのメソッドがコンストラクタであることを想定していたのか、他の人に疑問を抱かせることになります。

コンプライアンス違反のコード例

public class Foo {
   public Foo() {...}
   public void Foo(String label) {...}  // Noncompliant
}

コンプライアンス・ソリューション

public class Foo {
   public Foo() {...}
   public void foo(String label) {...}  // Compliant
}
