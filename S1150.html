注：このEnumerationインターフェースの機能は、Iteratorインターフェースと重複しています。さらに、Iteratorはオプションでremove操作を追加し、より短いメソッド名を持っています。新しい実装では、Enumeration よりも Iterator の使用を検討する必要があります。

コンプライアンス違反のコード例

public class MyClass implements Enumeration {  // Non-Compliant
  /* ... */
}

コンプライアンス・ソリューション

public class MyClass implements Iterator {     // Compliant
  /* ... */
}
