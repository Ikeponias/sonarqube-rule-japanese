hashCode とtoString は配列に対して利用できますが、ほとんど役に立ちません。hashCode は配列の「IDハッシュコード」を返し、toString はほぼ同じ値を返します。どちらのメソッドも、実際には配列の内容を反映した出力ではありません。 代わりに、関連する静的メソッドであるArrays に配列を渡す必要があります。

コンプライアンス違反のコード例

public static void main( String[] args )
{
    String argStr = args.toString(); // Noncompliant
    int argHash = args.hashCode(); // Noncompliant

コンプライアンス・ソリューション

public static void main( String[] args )
{
    String argStr = Arrays.toString(args);
    int argHash = Arrays.hashCode(args);
