Java 7以降、Stringsをswitch 引数として使用することができるようになりました。そのため、1つのString がif/else if 構造の3つ以上の値に対してテストされる場合、可読性を高めるために、代わりにスイッチに変換されるべきです。

なお、このルールは、プロジェクトのsonar.java.source が7 よりも低い場合、自動的に無効となります。

コンプライアンス違反のコード例

if ("red".equals(choice)) {  // Noncompliant
  dispenseRed();
} else if ("blue".equals(choice)) {
  dispenseBlue();
} else if ("yellow".equals(choice)) {
  dispenseYellow();
} else {
  promptUser();
}

コンプライアンス・ソリューション

switch(choice) {
  case "Red":
    dispenseRed();
    break;
  case "Blue":
    dispenseBlue():
    break;
  case "Yellow":
    dispenseYellow();
    break;
  default:
    promptUser();
    break;
}
